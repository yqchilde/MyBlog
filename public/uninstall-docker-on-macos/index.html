  • 情况二：打不开 Docker Desktop
  • 情况三：打不开 Docker Desktop 且情况二无法解决

情况一：可以打开 Docker Desktop

直接在 Mac 上打开 Docker Desktop，然后点击troubleshoot 进入后再点击 uninstall。

情况二：打不开 Docker Desktop

打开命令行执行以下命令：

1

/Applications/Docker.app/Contents/MacOS/Docker --uninstall
