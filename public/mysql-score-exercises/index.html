SELECT
	d.classid,
	a.sid,
	a.sname,
	SUM( c.score ) totalScore,
	COUNT( c.cid ) courseNum 
FROM
	t_student a,
	t_course b,
	t_score c,
	t_class d 
WHERE
	a.sid = c.sid 
	AND b.cid = c.cid 
	AND a.classid = d.classid 
GROUP BY
	a.sid

SELECT
	d.classid,
	a.sid,
	a.sname,
	SUM( c.score ) totalScore 
FROM
	t_student a,
	t_course b,
	t_score c,
	t_class d 
WHERE
	a.sid = c.sid 
	AND b.cid = c.cid 
	AND a.classid = d.classid 
GROUP BY
	a.sid 
HAVING
	( SUM( c.score ) > 180 )

	classid

1055错误坑

1055 - Expression #1 of SELECT list is not in GROUP BY clause and contains nonaggregated column 'd.sname' which is not functionally dependent on columns in GROUP BY clause; this is incompatible with sql_mode=only_full_group_by

解决方法：

  1. 在MySQL安装目录找到my.ini，在mysqld下面添加如下代码：

    sql_mode=STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION

  2. 重启mysql服务
